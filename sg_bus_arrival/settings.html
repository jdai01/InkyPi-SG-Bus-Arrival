Initialisation

To initialise this page, please following steps:
  1. Obtain API Key from LTA's Datamall.
  2. Go to API Keys page and enter your API key under LTA_DATAMALL_API_KEY and save entry.
  3. Go back to SG Bus Arrival page and click on "Update Now".
  4. Refresh current page and you should be able to access the settings.
